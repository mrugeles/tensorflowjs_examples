# Tensorflow JS Examples

## Install
```sh
npm install nodemon
npm install
```

## Run app

```sh
nodemon app.js
```

```sh
[nodemon] 2.0.8
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,json
[nodemon] starting `node .\app.js`
app started
```
Load app at http://localhost:8080/
